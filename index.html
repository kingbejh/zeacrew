📖 Panduan Sistem

🔐 Login & Role
👑 Owner: Akses penuh ke semua fitur termasuk edit data token & absensi
⚙️ Admin: Edit unit, absensi (tambah baru), gaji, pengajuan, staff
📋 Leader: Edit unit & staff, kelola absensi (tambah baru), approve pengajuan
📢 Marketing: Lihat unit, posting, ajukan pengajuan
👤 Staff: Lihat data sendiri, pengajuan, gaji sendiri
👥 Manajemen Staff
1. Buat akun dulu di "Kelola Akun"
2. Tambah data staff di "Staff"
3. Isi: Nama, WA, Gaji, Uang Makan/Hari, Jam Kerja, Libur Tetap
Note: Libur tetap = gaji full, tidak dipotong
🏠 Unit Apartemen
✅ READY = Siap ditempati
🔧 MAINTENANCE = Butuh perbaikan
⚠️ Kuning = Jatuh tempo < 7 hari
Klik unit untuk edit (Owner/Admin/Leader)
⚡ Token Listrik (Edit: Owner Only)
Catat setiap pembelian token listrik per unit.
Sistem otomatis deteksi unit BOROS jika > Rp 500.000/bulan
Fitur Edit & Hapus hanya untuk Owner - untuk koreksi kesalahan input data.
📋 Absensi (Edit: Owner Only)
Data dari mesin sidik jari, input oleh Admin/Leader
Potongan:
• 0-10 menit = Rp 0
• 11-30 menit = Rp 25.000
• >30 menit = 1 hari kerja
• IZIN/SAKIT/ALFA = 1 hari kerja
Fitur Edit & Hapus hanya untuk Owner - untuk koreksi kesalahan input data yang mempengaruhi perhitungan gaji.
💰 Penggajian
GAJI TRANSFER = Gaji Pokok - Semua Potongan

Uang makan TIDAK dikurangi (sudah diambil cash harian).
Slip gaji otomatis dikirim via WhatsApp.
